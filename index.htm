COPA Trade Kya Hai?

ITI COPA ek 1-saal ka vocational course hai jo Directorate General of Training (DGT) ke antargat aata hai. Iska uddeshya students ko computer hardware, software, aur programming basics mein maharat hasil karana hai.

Eligibility: 10th Pass
Duration: 1 Year (2 Semesters)

Kya Sikhenge Aap?

  • Computer Components aur Windows OS ka upyog.
  • MS Office Suite (Word, Excel, PowerPoint, Access).
  • Networking Concepts aur Internet.
  • HTML, CSS aur JavaScript (Web Designing).
  • Programming with Python / Java Basics.
  • Cyber Security aur E-commerce.
  • Tally (Accounting Software).
